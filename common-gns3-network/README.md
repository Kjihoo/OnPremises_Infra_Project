# 공용 — GNS3 네트워크 & 망분리

## 담당 역할
5인 공용 작업 (통합 단계에서 다같이 진행)

## 할 일
- GNS3 Router + L2 Switch 토폴로지 구성
- VLAN 3개 설정
  - VLAN10 (192.168.10.0/24) → DMZ (Nginx, Bastion)
  - VLAN20 (192.168.20.0/24) → Private (K8s 클러스터)
  - VLAN30 (192.168.30.0/24) → DB (MySQL)
- 방화벽 ACL 정책 적용
- VMware VM들과 GNS3 연동

## ACL 정책
| 출발 | 도착 | 포트 | 정책 |
|------|------|------|------|
| 외부 | VLAN10 | 443, 80 | 허용 |
| VLAN10 | VLAN20 | 30080 | 허용 |
| VLAN20 | VLAN30 | 3306 | 허용 |
| 외부 | VLAN20, VLAN30 | 전체 | 차단 |
| VLAN10 | VLAN30 | 전체 | 차단 |

## 시현
- 외부 → DB 직접 접근 → 차단
- 외부 → K8s 직접 접근 → 차단
- VLAN 망분리 증명

## 진행 시점
각자 파트 개발 완료 후 → 통합 직전에 다같이 진행
